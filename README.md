# JavaScript Number Operations

A JavaScript program that performs various operations on a given number, including finding the sum of its digits, checking whether it is an Armstrong number, checking whether it is a prime number, and printing all its factors.

## Objective

The objective of this assignment is to practice JavaScript fundamentals by implementing different number operations using loops, conditional statements, variables, and operators.

## Technologies Used

- HTML5
- JavaScript

## Project Structure

```text
JavaScript-Number-Operations/
│── index.html
│── script.js
└── README.md
```

## Features

- Check whether a number is prime.
- Print all factors of a number.
- Find the sum of digits of a number.
- Check whether a number is an Armstrong number.
- Display all results in the browser console.

## How to Run

1. Clone or download the repository.
2. https://github.com/kjwashim/Task-14-JavaScript-Loops.git
3. Open the project folder.
4. Open `index.html` in your web browser.
5. Press **F12** and open the **Console** tab to view the output.

## Sample Output

```text

Sum of its digits: 9
Is it an Armstrong number? Yes
Is it a prime number? No
Factors: 1, 3, 9, 17, 51, 153
```
